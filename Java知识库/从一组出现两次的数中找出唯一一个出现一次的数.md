```
// 异或法
        List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5, 5, 4, 2, 3, 1, 6);
        sort(numbers);
        int result = 0;
        int i = 1;
        for (Integer num : numbers) {
            System.out.println("第" + i + "个数: " + num);
            result ^= num;
            System.out.println("异或操作后的数: " + result);
            i++;
        }
```





先对数组进行排序，相同的数异或操作后为 0 ，任何数和 0  异或还是原来的数。
